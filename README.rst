Install
------------------------------------------------------------------------------

``rdso_ml`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install rdso_ml

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade rdso_ml
